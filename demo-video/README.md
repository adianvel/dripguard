# DripGuard demo video

30s monochrome product spot (1920×1080, 30fps).

## Scenes

| Time | Content |
|------|---------|
| 0–3s | DG mark + title |
| 3–8s | Problem — agents burn budgets |
| 8–19s | **Fund. Drip. Stop.** |
| 19–25s | Guardrails (VEST / CAP / ALLOW / VEL) |
| 25–30s | CTA — open the app |

## Render

```powershell
npm install
npm run render
# → out/dripguard-demo.mp4
```

Preview: `npm run studio`

Requires Chrome/Chromium (Remotion downloads if needed).
