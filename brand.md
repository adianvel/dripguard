# Brand — DripGuard

Status: active

## Positioning

**DripGuard** — rate-limited spend for autonomous agents.

Vesting-based spending governor on Stellar. Quiet, precise, technical.

## Voice

- Short sentences. Concrete guardrails.
- Prefer “max loss is designed” over “revolutionary AI.”
- Status language feels terminal-clean: `ACTIVE`, `PAUSED`, `CLAIMABLE`.

## Visual identity

**ASCII agent guard** + **simple whitespace**.

- Near-white canvas, near-black type, black CTAs
- Green only for success / vested progress
- Monospace for brand marks, addresses, amounts, status — not body prose
- No marketing color chrome, no gold/lavender spectacle

### Typography

- UI / display: Geist Sans
- Data / ASCII: Geist Mono
- Landing H1 large and calm; app body 14–15px

### Spacing

- Landing sections: large vertical air
- Content max width ≈1120px
- Prefer one column and thin borders

### Motion

Light only. Soft card hover, Lottie on in-view. Respect `prefers-reduced-motion`.
